# 搭建cnode后端平台
[[toc]]
## 准备工作

1. 登录服务器
```shell
ssh username@ip
```

2. 安装git
```
yum install -y git
```

3. 下载 NVM 源码并检查最新版本
```
git clone https://github.com/cnpm/nvm.git ~/.nvm && cd ~/.nvm && git checkout `git describe --abbrev=0 --tags`
```

4. 配置NVM环境变量
```
echo ". ~/.nvm/nvm.sh" >> /etc/profile
```

5. 读取环境变量

```
source /etc/profile
```
